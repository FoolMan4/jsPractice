     С репозиторием можно работать двумя способами:

     1. Через терминал в самом VS Code

    Команды для работы с репозиторием GitHub: 
        1. git status -> проверка статуса файлов (красный цвет -> измененные файлы, которые надо сохранить; зеленый цвет -> сохраненные файлы, которые можно отправлять на удаленный сервер)
        2. git add -A -> добавление изменений
        3. git commit -a -m"Title of the commit should be written here" -> создание точки изменений (нужны для того, чтобы откатить прогресс при случае ошибок в дальнейшем коде)
        4. git push -> отправление данных на удалённый сервер

    Ситуация: 

    Произведены изменения в коде
    Дальше прописываем команды в терминал поочередно: 
        git add -A 
        git commit -a m"Commit #1"
        git push
        git status (Проверить всё ли заработало)
